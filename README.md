# 🧪 Molecular Stereochemistry Analyzer

A professional Streamlit web application to analyze molecular structures and identify chiral centers using RDKit.

## 🚀 How to Host This Online (Free)

To get a shareable link that anyone can access, follow these 3 simple steps:

### 1. Upload to GitHub
1. Create a new repository on [GitHub](https://github.com/new).
2. Upload the following files to it:
   - `v2.py` (The main app code)
   - `requirements.txt` (List of dependencies)

### 2. Deploy to Streamlit Cloud
1. Go to [share.streamlit.io](https://share.streamlit.io).
2. Connect your GitHub account.
3. Click **"New app"** and select your repository and the `v2.py` file.
4. Click **"Deploy!"**

### 3. Share Your Link!
Once deployed, Streamlit will give you a unique URL (e.g., `https://your-app-name.streamlit.app/`) which you can share with anyone.

## 🛠️ Running Locally
If you want to test it on your computer first:
```bash
pip install -r requirements.txt
streamlit run v2.py
```

## ✨ Features
- **Visual Structure**: 2D rendering of molecules.
- **Automated Analysis**: Instant identification of chiral centers.
- **Configuration (R/S)**: Scientific determination of atom stereochemistry.
- **Presets**: Quick-load common molecules like Erythromycin or Glucose.
